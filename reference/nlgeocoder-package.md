# nlgeocoder package

The nlgeocoder package provides a R wrapper around the pdok webservice.

## Details

it allows to geocode data using the following functions:

- [`nl_geocode()`](https://urosconf.github.io/nlgeocoder/reference/nl_geocode.md):
  geocode a vector of addresses.

- [`nl_free()`](https://urosconf.github.io/nlgeocoder/reference/nl_free.md):
  find possible matches for one address.

- [`nl_suggest()`](https://urosconf.github.io/nlgeocoder/reference/nl_suggest.md):
  find possible suggestions for one address.

- [`nl_lookup()`](https://urosconf.github.io/nlgeocoder/reference/nl_lookup.md):
  lookup information of object found in `nl_suggest`.
